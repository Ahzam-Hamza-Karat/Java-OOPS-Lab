# Java OOPS Lab Experiments

This repository contains all Object-Oriented Programming (OOPS) lab experiments implemented in Java.

## 📘 Overview

These experiments demonstrate core OOPS concepts including:

- Classes and Objects
- Inheritance
- Polymorphism
- Encapsulation
- Abstraction
- Interfaces
- Exception Handling
- File Handling
- Collections
